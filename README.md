# FAP - Aponti | Automação e Testes de Software 🚀

Este repositório contém os projetos, cenários de testes e automações desenvolvidos durante o **FAP (Programa de Aceleração em Programação)** da **Aponti**, com foco em Garantia de Qualidade (QA) e Testes de Software.

O objetivo deste projeto é aplicar as melhores práticas de mercado para garantir a confiabilidade, segurança e performance de aplicações reais.

---

## 🛠️ Tecnologias, Ferramentas e Frameworks
*As ferramentas utilizadas variam conforme o módulo do projeto:*

*   **Testes de API:** Postman / RestAssured
*   **Automação Web/Mobile:** Cypress / Selenium / Playwright
*   **Linguagens:** JavaScript / TypeScript / Java
*   **Gerenciamento de Testes:** Jira / Trello / Markdown (Planilhas de Casos de Teste)
*   **CI/CD:** GitHub Actions
